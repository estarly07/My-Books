# My Books

Book  es un proyecto que permite guardar tus libros de estudio para tener un acceso rápido y facíl 📚.

<h3>Características</h3>

  - Lenguaje de programación Kotlin.
  - Implementación de patron Room. 
  - Arquitectura MVVM.
  - Binding.
  - Servicios.
  - Cloud Firestore.☁️
  - Dagger Hilt.💉
